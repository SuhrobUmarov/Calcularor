Description for working buttons:

Button "7" - Number 7
Button "8" - Number 8
Button "9" - Number 9
Button "/" - Division
Button "4" - Number 4
Button "5" - Number 5
Button "6" - Number 6
Button "*" - Multiplication
Button "1" - Number 1
Button "2" - Number 2
Button "3" - Number 3
Button "-" - Subtraction
Button "0" - Number 0
Button "." - Decimal Point
Button "+" - Addition
Button "=" - Equal Sign
Button "C" - Clear All

Also, you can edit the text bringing your mouse to the writing field and pushing there, you can start typing the numbers

List of buttons that are not implemented:

mc   Memory clear
mr   Memory recall
m-   Memory minus
m+   Memory plus
CE   Clear entry
√x   Square root
%   Percentage
π   pi = 3.1415926536
xy   Exponent
R2   Round to 2 decimals (cents)
R0   Round to 0 decimals (dollars)
